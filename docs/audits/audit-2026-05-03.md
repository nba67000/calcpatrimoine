### DIVERGENCE MAJEURE - Seuil 150 000€ absent

**Problème** : Pour un contrat > 8 ans avec versements **après 27/09/2017**, le code applique systématiquement le PFU à 12,8 % (soit 30 % tout compris). Or la règle en vigueur (Art. 125-0 A CGI, confirmé impots.gouv.fr 2026-03-20) est :
- Encours total ≤ 150 000€ → PFU **7,5 %** → total **24,7 %**
- Encours total > 150 000€ → PFU **12,8 %** → total **30 %**

**Impact** : Tous les utilisateurs avec un contrat > 8 ans, versements uniquement post-2017, et un encours ≤ 150 000€ voient un impôt surestimé. C'est le cas le plus courant (épargnant moyen).

**Correction proposée** (attente validation Nicolas) :
1. Ajouter un champ `encoursTotalContrats` dans `AssuranceVieInputs` (montant total des versements nets sur tous les contrats AV).
2. Dans `calculerTauxPFU`, pour les contrats > 8 ans, calculer le taux applicable selon le seuil :
   - Si `encoursTotalContrats <= 150_000` : taux IR = 7,5 % pour la totalité des gains post-2017
   - Si `encoursTotalContrats > 150_000` : prorata 7,5 % / 12,8 % selon la fraction au-delà de 150k€

**Fichier à modifier** : `src/lib/assuranceVie.ts` + `src/types/assuranceVie.ts` + `src/components/Calculator/AssuranceVieCalculator.tsx`
